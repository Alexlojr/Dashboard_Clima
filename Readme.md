# Dashboard de Clima – Python + CustomTkinter

Este projeto é um **Dashboard de Clima** desenvolvido em Python utilizando **CustomTkinter**, **Pillow**, **Requests** e API da https://openweathermap.org/api/one-call-3.  
O app permite consultar informações meteorológicas de forma simples e visual, com ícones e interface amigável.

![img_1.png](imgs/img_1.png)

---

## Funcionalidades

- Busca informações de clima via API
- Exibe temperatura atual
- Mostra condições climáticas com ícones (Calor ou Frio)
- Interface visual com CustomTkinter
- Estrutura organizada em arquivos (`Main.py` + `Classes_Functions.py`)

---

## Instruções de Uso

- git clone https://github.com/Alexlojr/Dashboard_Clima.git
- pip install -r requirements.txt
- **Importante:** Para funcionar, você deve adicionar sua chave de API da OpenWeather no arquivo:`Classes_Functions.py`

![img.png](imgs/img.png)
- python Main.py
